# Backend Projects

Este repositório reúne projetos e exercícios pessoais focados no **desenvolvimento Back-End**.

Os códigos foram desenvolvidos com o objetivo de praticar **lógica de programação, manipulação de dados e implementação de funcionalidades comuns em aplicações back-end**, simulando cenários utilizados no desenvolvimento de sistemas.

## Tecnologias

- C#
- Python

## Conteúdos abordados

- Lógica de programação
- Manipulação de dados
- Estruturas de controle
- Criação de funcionalidades de back-end
- Organização de código

## Objetivo

Praticar e consolidar conhecimentos em **desenvolvimento back-end**, utilizando linguagens de programação e aplicando conceitos fundamentais para construção de aplicações e serviços.
